Cocos2d-html5 download builder tools
==================

download builder打包工具。(mac下测试，其他平台未测试)

安装:

```
npm install -g cocos-builder
```

必须全局安装。

使用方法:

进入cocos2d-html5引擎文件夹内。运行cocosbuilder。

接受两个参数：

--debug

--name moduleName

```
cocosbuilder --debug --name v3.0RC3
```

版本对应:

0.0.7 --- v3.0 rc3
